# 🎨 Clínica & Escola App - Frontend

Interface de utilizador moderna e responsiva, construída em **React** com **Vite**, focada na experiência do aluno e gestão administrativa.

## 🚀 Tecnologias
- **React 19+**
- **Vite** (Build tool)
- **Tailwind CSS** (Estilização)
- **Axios** (Cliente HTTP)
- **React Hook Form + Zod** (Validação de formulários)
- **Lucide React** (Ícones)

## 🛠 Configuração Inicial

1. **Entrar na pasta:** `cd frontend`
2. **Instalar dependências:** `npm install`
3. **Configurar Ambiente:** Criar ficheiro `.env`

   VITE_API_URL=http://localhost:8000/api
